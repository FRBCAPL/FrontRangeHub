              How to Play

              Quick Start

              1. 1 INSERT 2 QUARTERS
              2. 2 Find the game number on the cabinet menu (or search here)
              3. 3 Use the joystick to highlight the game, then press START
              4. 4 Beat a high score and enter your initials on the cabinet

              On the Cabinet

              • Joystick up / down — Move one game at a time through the menu
              • Joystick left / right — Jump ahead 10 games at a time (scroll faster)
              • START button — Launch the highlighted game
              • Hold Player 1 button — Exit back to the main game menu
              • High score screen — Enter your 3-letter initials when prompted

              Using This Screen

              • Search Games — Look up a game by name and see its number
              • High Scores — View leaderboards
              • Submit Score — Photo must show the new score clearly (face optional). Selfie with your face? We will use that photo on the leaderboard!
              • Menu picture — Tap the game list image at the bottom to see the full cabinet menu
